Informatie-Aan-Zee Demo

Demo voor Informatie Aan Zee ontwikkeld door PACKED vzw i.s.m. het Museum voor Edelsmeedkunst, Juwelen en Diamant.

Deze demo maakt gebruik van de publieke API van het collectiebeheersysteem van het MEJD. Dit systeem is gebaseerd op Qi van Keepthinking.

De demo is geschreven in Javascript en HTML5. Het framework is AngularJS. De API-calls worden omgeleid via een PHP-wrapper om het CORS-probleem te omzeilen.

Alle code is beschikbaar op Github als GPLv2.
